Hints

Push: You can push a box forward by going behind it and then advance againt it. However, a box can only be pushed if the square in which it will be pushed to is empty.

Pull You can't pull a box per say, but what you can do is go behind it and push it towards the desired square.

Push Double You can't push 2 boxes at the same time.

Targets The green dots present on some squares are called targets. The player can't interact with targets. Their purpose is to recieve boxes, one box per target. A box is correctly placed on a target when its color change.

Win the game In order to win the game, you must place all the boxes on targets, one box per target, in as few moves as possible.
